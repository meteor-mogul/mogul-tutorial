meteor-mogul tutorial: Table of Contents

Lesson 1: Creating a new meteor-mogul app

In this lesson you will

  1. create your first meteor-mogul app
  2. practice using git to download and manage source code
  3. run your first meteor-mogul app
  4. fork a GitHub project so you can maintain your own repo

Preliminaries

If you haven't already, install Meteor and git. You'll also need a programming editor (I recommend Atom). Optionally, you might want to install GitHub Desktop, which makes it easier to manage source code on GitHub. Finally, you'll also need to set up a GitHub account for yourself.

Getting meteor-mogul Apps

All meteor-mogul apps are on GitHub. Start by using git to clone the meteor-mogul tutorial.

This creates your own local copy of the master branch of the tutorial repo.

$ git clone https://github.com/meteor-mogul/mogul-tutorial.git
$ cd mogul-tutorial

Note two things:

  1. any code line that starts with a $ means a command you type in your terminal
  2. all the following terminal commands assume you are executing them from the mogul-tutorial directory

Checkout the Branch for this Lesson

As you work your way through this tutorial, you'll be changing the code in the mogul-tutorial app.

A git repo contains not just one version of code, but many versions. Sets of changes in code are called branches. By default, when you clone a repo with git, you get the master branch. That is usually the branch that a project is ready to share with the public.

For this tutorial, for each lesson we've created a branch in our mogul-tutorial repo.

To get just the code that should be in the app at this point, checkout the lesson01 branch.

$ git checkout lesson01

To go back to the master branch:

$ git checkout master

Can you see what's different between the two branches? You can practice going back and forth between the two branches (if you have GitHub Desktop installed, try figuring out how to use it to switch between branches, too). Make sure you end up with the lesson01 branch checked out before you proceed.

Running meteor-mogul Apps

To run the mogul-tutorial app, first make sure you've got a lesson branch checked out (unlike most other meteor-mogul apps, this tutorial's master branch is not intended to run as an app). Then do the following in your terminal:

$ meteor npm install
$ meteor

Open a web browser to http://localhost:3000. You should see the meteor-mogul mogul-tutorial running.

To stop the app, do:

$ [Press Control-C]

If you refresh your browser, you should see that the app has stopped running.
