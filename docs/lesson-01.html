Lesson 1: Creating a new meteor-mogul app

If you haven't already, install Meteor and git. You'll also need a programming editor (I recommend Atom). Optionally, you might want to install GitHub Desktop, which makes it easier to manage source code on GitHub.

All meteor-mogul apps are on GitHub. Start by using git to clone the meteor-mogul tutorial. This creates your own local copy of the master branch of the tutorial repo.

To practice creating and running a meteor-mogul app, open your terminal and type:

$ git clone https://github.com/meteor-mogul/mogul-tutorial.git

Note that any code line that starts with a $ means a command you type in your terminal.

To run the mogul-tutorial app, do the following in your terminal:

$ cd mogul-tutorial
$ meteor npm install
$ meteor

Open a web browser to http://localhost:3000. You should see the meteor-mogul mogul-tutorial running.

To stop the app, do:

$ [Press Control-C]

If you refresh your browser, you should see that the app has stopped running.
