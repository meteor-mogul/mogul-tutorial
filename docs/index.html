meteor-mogul tutorial

In this tutorial, you create a simple meteor app to manage a 'to do' list and collaborate with others on those tasks. By the end, you will have a basic understanding of meteor-mogul.

By working through this tutorial, you'll not only learn how to use Meteor, but you'll also learn a little bit about VueJS, a view layer, and Bootstrap, a CSS framework.

Along the way you'll practice using GitHub and git to get and manage source code.
